# Python

Install package:

```bash
pip install PKG_NAME
```

Alternatively, if the paths are not working correctly:

```bash
python -m pip install PKG_NAME
```

Upgrade package:

```bash
pip install --upgrade PKG_NAME
```

Uninstall package:

```bash
pip uninstall PKG_NAME
```

Run script:

```bash
python FILENAME.py
```

Install pip package from folder:

```bash
pip install .
```
